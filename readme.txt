== Plugin Description ==
See all available options and their description on plugin's settings page. A summary:

	* Add featured image column in dashboard posts,pages lists ( https://i.imgur.com/XOg9BIP.png )
	* Ability to Sort pages by parent (in dashboard)
	* Enable Search Highlighting
	* Allow exact search
	* Enable custom admin css
	* Enable custom frontend css
	* Auth cookie expiration
	* Dashboard Search items amount
	* Posts per page amount
	* Editor permissions increase
	* Override excerpt length
	* Override excerpt text
	* Noindex garbage pages
	* Extra mimetypes
	* Enable custom post types
	* Remove admin topbar
	* Redirect to subdirectory
	* Override wordpress mail sending settings
	* Remove emojis styles & scripts
	* Restore jQuery <b>$</b> definition in wp-head loading
	* Disable update notifications
	* Disable WP rest api
	* Random favicons during development
	* Add body css classes
	* Add content & excerpt css classes
	* Add category title field
	* Enable buffer text replacement
	* Save a note
	* feature to insert Google Analytics (tag) script
	* Increase maximum upload size
	* Increase maximum memory limit

== Installation ==
A) Enter your website "Admin Dashboard > Plugins > Add New" and enter the plugin name
or
B) Download plugin from WordPress.org , Extract the zip file and upload the container folder to "wp-content/plugins/"
